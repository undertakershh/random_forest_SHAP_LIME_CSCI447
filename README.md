# random_forest_SHAP_LIME_CSCI447
This repository contains the code for the Machine Learning Course (CSCI 447) project on the topic "Explainability Case Study Pick one model + dataset and compare SHAP/LIME; test explanation stability under perturbations."

The code requires Python 3.10 to be installed, and libraries such as LIME and SHAP installed via pip. Alternatively, the code can be run via Google Colab, as the creating team used the Google Colab to implement it.
